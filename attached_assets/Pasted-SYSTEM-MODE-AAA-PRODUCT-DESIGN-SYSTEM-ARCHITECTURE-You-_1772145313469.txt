INFRASTRUCTURE SYSTEM

Resources:

- Cash
- Hash Units
- Power Capacity
- Cooling Capacity
- Wear Level
- Security Level
- Insight (research currency)

Production Formula:

Hash Output =
Σ (rigHashrate × powerEfficiency × coolingEfficiency × uptime)

Uptime =
1 - wearPenalty - incidentPenalty

Constraints:

- If power exceeds cap → efficiency drops
- If heat exceeds cap → incident probability rises
- If wear ignored → uptime collapses

This creates natural infrastructure scaling tension.
